Dashboard Interactivo de Comisiones — Telesales

Power BI

Seguimiento visual año a la fecha de comisiones y porcentaje de logros por área. Permite filtrar por supervisor, mes y ciclo. Muestra KPIs por agente con gráficos comparativos de adherencia, conversión y Customer Experience.

Dashboard Billing — Indicadores en Tiempo Real

Automatización

Reportes de cobros que antes se generaban manualmente una vez al mes, ahora se actualizan diariamente de forma automática. Incluye vista de 12 meses, seguimiento de cierres y productividad, accesible desde SharePoint.

Stack: Power BI · Python · SQL · PAD · Power Query Impacto: De reporte mensual a actualización diaria

Collections Performance — Seguimiento de Cobros

BI

Herramienta de seguimiento diario a la productividad del área de cobros. Muestra monto asignado vs recuperado, logro por empleado y ciclo. Acceso online desde SharePoint, reduciendo dead time de los colaboradores.

Stack: Power BI · VBA · Python · SQL Impacto: Seguimiento en tiempo real diario

Employee Performance Report — Promociones Internas

App

Panel automatizado de desempeño individual para evaluación de promociones internas. Consolida 3 fuentes de datos en un solo panel, mostrando logro general, PRI y mix fijo-móvil por mes. Redujo tiempos de respuesta en un 90%.

Stack: Power BI · SQL · Power Query Impacto: -90% tiempo de respuesta · 3 fuentes unificadas

Cases Track — Control SLA para Área de Créditos

Automatización

Sistema de seguimiento SLA en tiempo real para el área de créditos. Muestra casos abiertos, próximos a vencer y vencidos por segmento. Genera alertas automáticas cada 5 minutos para prevenir vencimientos y garantizar el cumplimiento del SLA.

Stack: Power BI · SQL · Power Automate Impacto: Alertas automáticas · Control de vencimientos en tiempo real
